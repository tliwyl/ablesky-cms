
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8081
npm run dev

# build for production with minification
npm run build

```

# Folder structure
* build - webpack config files
* config - webpack config files
* dist - build
* src -
    * api
    * assets
    * common
    * components 
    * mock
    * styles
    * views 
    * vuex
    * App.vue
    * main.js 
    * routes.js
* static - static assets
